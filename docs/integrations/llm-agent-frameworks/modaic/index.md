[Modaic](https://www.modaic.dev/) is a hub to manage and share DSPy programs.

## Modaic and Weaviate
Weaviate search can be used as a tool with agents published on Modaic Hub. 

Learn about the [Modaic SDK](https://docs.modaic.dev/) and check out the programs on the hub [here](https://www.modaic.dev/programs).
