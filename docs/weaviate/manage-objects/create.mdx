## Generate deterministic IDs

You can generate an ID based on your data object.

:::info
Object IDs are not randomly generated. The same value always generates the same ID.<br/>Weaviate throws an error if you provide a duplicate ID. Use deterministic IDs to avoid inserting duplicate objects.
:::
