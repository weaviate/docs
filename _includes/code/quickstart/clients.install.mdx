import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs className="code" groupId="languages">
<TabItem value="py" label="Python">

Install the latest, [Python client `v4`](/weaviate/client-libraries/python), by adding `weaviate-client` to your Python environment with `pip`:<br/><br/>

```bash
pip install -U weaviate-client
```

</TabItem>
<TabItem value="ts" label="JavaScript/TypeScript">

Install the latest, [JS/TS client `v3`](/weaviate/client-libraries/typescript), by adding `weaviate-client` to your project with `npm`:<br/><br/>

```bash
npm install weaviate-client
```

</TabItem>
<TabItem value="go" label="Go">

Add `weaviate-go-client` to your project with `go get`:<br/><br/>

```bash
go get github.com/weaviate/weaviate-go-client/v5
```

</TabItem>
<TabItem value="java6" label="Java v6 (Beta)">

Add this dependency to your project:<br/><br/>

```xml
<dependency>
  <groupId>io.weaviate</groupId>
  <artifactId>client6</artifactId>
  <version>6.0.0-M1</version> <!-- Check latest version: https://github.com/weaviate/java-client -->
</dependency>
```

</TabItem>
<TabItem value="java" label="Java">

Add this dependency to your project:<br/><br/>

```xml
<dependency>
  <groupId>io.weaviate</groupId>
  <artifactId>client</artifactId>
  <version>4.8.3</version>  <!-- Check latest version: https://github.com/weaviate/java-client -->
</dependency>
```

</TabItem>
<TabItem value="csharp" label="C# (Beta)">

Add this package to your project:<br/><br/>

```xml
<PackageReference Include="Weaviate.Client" Version="0.0.1-beta.4" />   <!-- Check latest version: https://github.com/weaviate/java-client -->
```

</TabItem>
