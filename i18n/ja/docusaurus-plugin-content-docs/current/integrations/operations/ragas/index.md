Ragas は、検索拡張生成 ( RAG ) アプリケーションの評価を支援するフレームワークです。

## Ragas と Weaviate
Ragas のメトリクスは次のとおりです:
* `faithfulness`
* `answer_relevancy`
* `context_precision`
* `context_recall`

Ragas を使用するには、`question`、`answer`、`ground_truths`、`contexts` を JSON オブジェクトに保存して Ragas に送信する必要があります。
