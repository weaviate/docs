TruLens とは、 LLM アプリケーションを評価および追跡するためのオープンソースプロジェクトで、アプリを体系的に評価、検査、改善できます。

## TruLens と Weaviate
TruLens は、 Weaviate を活用するアプリケーションの評価およびトレースに利用できます。これには Weaviate Query Agent への呼び出しのトレースもサポートされています。
