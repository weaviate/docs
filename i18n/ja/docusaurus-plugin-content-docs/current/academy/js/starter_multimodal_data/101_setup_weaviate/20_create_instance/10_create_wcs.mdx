ここでは、Weaviate Cloud (WCD) インスタンスを作成します。WCD はクラウドで稼働する完全マネージドの Weaviate インスタンスで、インストールや保守が不要なため Weaviate を始めるのに最適です。

### <i class="fa-solid fa-chalkboard"></i> WCD Console にログインする

[WCD Console](https://console.weaviate.cloud/) にアクセスし、認証情報でログインします。まだアカウントをお持ちでない場合は、ログイン画面の <kbd>Register here</kbd> リンクからサインアップできます。

### <i class="fa-solid fa-chalkboard"></i> Weaviate インスタンスを作成する

コンソールで Dashboard に移動し、<kbd>Create cluster</kbd> ボタンをクリックします。次の画面で以下を設定します。

- 「Free sandbox」タブを選択
- クラスター名を入力
- 「Enable authentication」を「Yes」に設定

<kbd>Create</kbd> ボタンをクリックして Weaviate インスタンスを作成します。完了まで数分かかります。

### <i class="fa-solid fa-chalkboard"></i> Weaviate インスタンスの詳細を取得する

インスタンスが作成されたら、<kbd>Details</kbd> ボタンをクリックして詳細を確認できます。クラスター URL と API キーを探してください。

これらの情報は Weaviate インスタンスへ接続する際に必要です。

Weaviate Cloud (WCD) インスタンスへ接続するには、クラスター URL と API キーが必要です。これらは WCD Console で確認できます。
