### <i class="fa-solid fa-code"></i> 接続の終了

Weaviate クライアントの使用が終わったら、接続を終了する必要があります。これによりリソースが解放され、接続が正しくクローズされます。

ベストプラクティスとして、 `try`-`finally` ブロックの使用を推奨します。簡潔にするため、以降のコードスニペットでは `try`-`finally` ブロックを省略します。
