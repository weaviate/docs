ここでは、Weaviate Cloud ( WCD ) インスタンスを作成します。WCD はクラウド上で稼働する完全マネージド型の Weaviate インスタンスです。インストールやメンテナンスが不要なため、Weaviate を始めるのに最適な方法です。

### <i class="fa-solid fa-chalkboard"></i> WCD コンソールにログインする

[WCD コンソール](https://console.weaviate.cloud/) にアクセスし、認証情報でログインします。まだアカウントをお持ちでない場合は、ログイン画面の <kbd>Register here</kbd> リンクから登録できます。

### <i class="fa-solid fa-chalkboard"></i> Weaviate インスタンスを作成する

コンソールで Dashboard に移動し、<kbd>Create cluster</kbd> ボタンをクリックします。次の画面で以下を設定してください。

- 「Free sandbox」タブを選択
- クラスター名を入力
- 「Enable authentication」を「Yes」に設定

<kbd>Create</kbd> ボタンをクリックすると、Weaviate インスタンスが作成されます。完了まで数分かかります。

### <i class="fa-solid fa-chalkboard"></i> Weaviate インスタンスの詳細を取得する

インスタンスが作成されると、<kbd>Details</kbd> ボタンをクリックして詳細を確認できます。クラスター URL と API キーを確認してください。

これらの情報は Weaviate インスタンスへ接続する際に必要です。
