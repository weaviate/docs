### <i class="fa-solid fa-code"></i> 接続の終了

Weaviateクライアントの使用が終わったら、接続を終了するようにしてください。これによりリソースが解放され、接続が適切にクローズされます。

ベストプラクティスとしては `try`-`finally` ブロックを使用することを推奨します。簡潔にするため、以降のコードスニペットでは `try`-`finally` ブロックは省略します。
