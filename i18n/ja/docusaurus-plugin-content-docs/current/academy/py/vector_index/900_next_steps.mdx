## インデックス選択

インデックス タイプを選択する際は、次を目安にしてください。

- **Flat index** : サイズが既知の小規模コレクション向け。  
- **HNSW index** : サイズが既知の大規模コレクション向け。  
- **Dynamic index** : サイズが不明、または時間とともに増加する可能性があるコレクション向け。

また、マルチテナント環境では、`dynamic` インデックスをデフォルトにすると便利です。これにより、一部のテナントは `flat` インデックスのままにし、成長したテナントは自動で `hnsw` に変換できます。
