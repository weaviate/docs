ようこそ！

ここから、Weaviate とともにベクトルの世界を旅していきます。このユニットでは、Weaviate の基本について概観します。

まず、Weaviate が何であり、何ができるのかを理解します。次に、ベクトルデータベースとベクトル検索について学んだ後、実際に Weaviate を起動し、ベクトル検索を実行します。

このユニットを終える頃には、コース全体をスムーズに学習し、さらに自身のプロジェクトで Weaviate を活用するための強固な基盤が身につきます。
