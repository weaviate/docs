オブジェクトを Weaviate にインポートする際、次のいずれかを選択できます。  
- **オブジェクトのベクトルを提供** して Weaviate に渡す  
- **ベクトライザーを設定** してインポート時にベクトルを取得する  
- **これらを組み合わせる** ことで、ベクトライザーを設定しつつ明示的にベクトルを提供する  

この **5 分間の動画** では、3 つのワークフローすべてを Python のコード例とともに概観します。

### 前提条件

この動画を見る前に、[Quickstart チュートリアル](docs/weaviate/quickstart/index.md) を一通り試すか、Weaviate のスキーマ、インポート、クエリの基本に慣れておくことをおすすめします。

### Jupyter notebook とデータ

対応する Jupyter notebook とデータは [こちらのリポジトリ](https://github.com/weaviate-tutorials/vector-provision-options) にあります。
