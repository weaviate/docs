オブジェクトを Weaviate にインポートする際、次の方法を選択できます。  
- **オブジェクト ベクトルを提供** して Weaviate に登録する  
- **ベクトライザーを設定** してインポート時にベクトルを取得する  
- **これらを組み合わせる** ことで、ベクトライザーを設定しつつ明示的にベクトルを提供する

この **5 分間のビデオ** では、3 つのワークフローすべてを Python のコード例とともに概観します。

### 前提条件

ビデオを視聴する前に、または Weaviate のスキーマ、インポート、クエリの基本に慣れておくために、[クイックスタート チュートリアル](docs/weaviate/quickstart/index.md) をご覧いただくことを推奨します。

### Jupyter notebook & データ

対応する Jupyter notebook とデータは [こちらのリポジトリ](https://github.com/weaviate-tutorials/vector-provision-options) で入手できます。
